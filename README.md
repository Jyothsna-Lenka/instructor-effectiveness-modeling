# Instructor Effectiveness Modeling

This project analyzes instructor performance on an EdTech platform using batch-level learning data.

## Objectives
- Define instructor effectiveness using learning metrics
- Aggregate batch data to instructor-level
- Train a machine learning model to predict effectiveness tiers
- Interpret model results and provide business insights

## Techniques Used
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Random Forest Classification
- Stratified Cross Validation
- Model Evaluation

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
